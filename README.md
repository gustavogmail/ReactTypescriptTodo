# Welcome to React Todo list app (TS) 👋

> Simple Todo list app built with React hooks and TypeScript.

### 🏠 [Homepage](https://github.com/gustavogmail/ReactTypescriptTodo#readme)

## Prerequisites

- node >=11.0.0
- npm >=6.0
- yarn >=1.16.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
